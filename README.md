# 📈 Tata Motors Stock Analysis (FY26)

This project provides a visual overview of Tata Motors' stock trend and revenue segment mix for FY26. It is intended for learning financial analysis and data visualization.

## 📊 Overview

The project includes:

- 52-week stock price trend
- Current Market Price (CMP)
- 52-week High and Low
- FY26 Revenue Segment Mix
- Key business insights
